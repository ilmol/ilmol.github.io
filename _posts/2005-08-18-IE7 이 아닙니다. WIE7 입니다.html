--- layout: post comments: true title: IE7 이 아닙니다. WIE7 입니다 date: 2005-08-18 00:28:23.000000000 -04:00 categories: - BROWSER ---

BetaNews | Microsoft Rebrands Internet Explorer 7

마소의 슬슬 전략이 얼굴을 들어내고 있습니다. 위의 기사가 이야기 해 주듯 인터넷익스플로러7(Internet Explorer 7) 은 단순히 IE7 이 아닌, Windows Internet Explorer 7 이라고 못박으며 이 새로운 브라우저의 이름을 바꾸었습니다. 아무리 무료 브라우저라고 하지만 Windows Vista 라는 차기 마이크로소프트(Microsoft) 의 새 OS 안에 종속 되어 있다는걸 재확인 시키는 작업같습니다.

물론 이뿐만이 아닌 Windows Vista 에 첨가되어 나올 WIE7 은 출시된지 얼마되지않은 새 로고 또한 약간 변형되어 나올꺼라고 하네요. Windows 2k 이하 는 지원되지도 않으니 옛날처럼 Windows Vista 를 사용하게 하는데 혹은 WIE7을 사용하게 하는데 서로가 필요하게 만들어 버리는 방향으로 나아가려고 하는게 뚜렷히 보입니다. 이미 마소에서 아직까지 비스타가 대부분의 컴퓨터에서 느리게 돌아간다는걸 시인한 상황에다 여러 공격들, 불매운동들이 벌어지고 있는 상황에서 어떻게 해쳐나가게 될지. 어떻게는 해낼테지만 말입니다.
