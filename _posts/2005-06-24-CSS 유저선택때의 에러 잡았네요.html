예전에 피오넬님께서 고생고생 하시면서 겪었던 문제를 이제서야 잡았습니다. 피오넬님댁의 TT 버전을 보시면 유저가 원하는 스킨을 고를수 있게 되어있지만, 원래의 스킨 말고 다른 옵션 스킨을 선택할시에, 즉 다른 css 파일을 선택할 시에 새롭게 로딩될때마다 잠시 원본스킨이 자꾸 로딩되었었습니다.

문제는 간단합니다. 바로 컴파일때 위에서 아래로 읽는 순서에 의해서 javascript를 먼저 읽어주기때문에 그랬더군요. 한마디로 <script> 부분이 스타일시트지정 link 부분보다 밑으로 내려와야 하는것입니다.

이부분에서 script 를 <link> 밑으로 내려주어야 하는것이죠.

이제 잡았습니다.

혹시 테터라서 안되었나 라고 생각하셨던 분들 안심하시고 스타일시트 changer 를 적용하셔도 될듯합니다.
