CSS Reboot 은 전세계 여러 옛날제작방식의 웹사이트들을 CSS 를 사용하여 웹표준에 맞게 제작하여 신청하는 이벤트 입니다. 더 자세한 내용은 제 옛날 글을 참고하시면 되겠습니다. 예전과는 다르게 상당히 사이트도 많이 발전하였고, 스폰서도 이제 제대로 받나봅니다.

유명한 9rules 의 Reboot 참여 인터뷰가 있네요. 신청하시는 분들 보고 링크합니다. Pilza N.com
