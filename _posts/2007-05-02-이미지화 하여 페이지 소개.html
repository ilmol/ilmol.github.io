최근에 테마를 제대로 넣으면서 많은 조정을 하고 있는 중입니다. 기본 틀을 제대로 잡아주니 무엇을 하던간에 편하네요. 아무튼 워드프레스로 글(word)을 찍어내시는(press) 분들 중 워드프레스의 꽃이라 불리우는 페이지 기능을 쓰시는 분들이 많은 것으로 압니다. 혹시나 만드신 페이지들을 좀더 문서답게 이미지화 하여 소개하고 싶으시면 이러한 방법을 쓰실수 있겠네요. style.css 에 아래 css 를 추가하시고 {% highlight css %} .page_document {float:left; width: 150px; height:220px; background: url(이미지 주소) no-repeat top left; margin: 0 15px 10px 10px; }

/* 문서 제목 부분 */ .page_document a {float:left;width: 149px; height: 140px; text-align:center; font-family: Georgia, serif; font-size:16px; font-weight:bold; color:black; padding-top: 40px; }

/* 마우스 오버일때 효과 */ .page_document a:hover{background:none;color:#951a14;} {% endhighlight %}

그리고 페이지를 소개하고 싶으신 곳에 "page_document" 이라는 div 안에 링크와 설명을 넣으면 되겠네요 {% highlight html %}

About Me나에대한 이야기 이다

{% endhighlight %}
