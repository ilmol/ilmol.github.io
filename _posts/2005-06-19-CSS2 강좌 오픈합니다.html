음.. 예정보다 너무 많이 늦어져서 죄송합니다. C.D.K. 블로그와 함께 나누었으면 좋겠다는 생각을 했지만 블로그의 준비가 약간 늦어지고 있는 시점이네요. 모든분들이 열심이시니 감사히 기다리고 있습니다. 기대도 되구요 ㅎ

곧 열리게 되면 지윤님과 상의해서 CSS 기초정리를 더 보강하려고 합니다. 물론 CDK 블로그에서는 기초적인 부분 보다는 실제 적용 팁들, 문제들을 다루며 토론하는 장소가 될것이지만 그래도 언제나 기초가 가장 중요한 부분이니 조심스럽게 더 다듬어야 한다고 봅니다. 후니님께서 XHTML 부분인 구조적인 사고 부분을 준비하신다고 하셨으니 그부분 많이 많이 많이 기대하셔도 될듯합니다. 역시 쵝오.

음 강좌 라고 하기는 그렇고 CSS 이해를 돕기위한 포스팅들은 세부분입니다. 절대 거창하지 않죠

오히려 "에게?" 하실수도 있으시겠네요. CSS 는 실전입니다. 이해하는건 금방이라고 봅니다. 아무튼 그 세부분을 보면

  1. CSS 문법, SELECTER 부분인 이름지정하기
  2. 예제를 통한 이해돕기 부분
  3. CSS2 사전

이며, 이 세가지만 잘 이해하셔도 CSS2를 쓰시는데는 커다란 문제가 없을것으로 보입니다.

특별히 잘못 설명한 부분이나 질문들을 받을수 있도록 각 페이지에 코멘트 부분을 열어놓았습니다. 지적도 바라고 건의도 부탁드립니다. 오타도 많고 실수한 곳도 있을테니 몇일간은 긴장하고 지켜보겠습니다 ㅎㅎ (이런부분을 보면 베타라고 부르고 싶네요 크흑.... ㅡㅜ)
